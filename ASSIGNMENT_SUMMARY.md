# Jiva Health Assignment Summary

The PDF asks for a React/Next frontend for a Jiva Health admin dashboard based on the provided Figma references.

## Required screens and features covered

- Dashboard with summary cards for total users, Prime users, orders, and total spend
- User Management page with registered users, role, status, joined date, spend, and actions
- User Detail page with personal information, contact details, DOB, gender, blood group, address, Prime status, and active status
- Family Members section with add, edit, and delete workflows
- Order History with order ID, date, item names, quantity, amount, status, and payment method
- Payment History with transaction ID, order link, amount, method, and payment status
- Appointment section for booked and completed health appointments
- Search and filters for user management
- Responsive layout for desktop and mobile
- Add User modal
- Clickable Order Detail page with itemized medicine list, shipping address, and payment details
- React + TypeScript source with Vite setup
